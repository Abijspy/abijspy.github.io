Project Overview

The Chatbot uses a combination of simple NLP and pattern-matching to interpret user queries. The user interacts with a Tkinter GUI; responses are driven by a JSON file containing questions and pattern-based replies.

How it works

  • User types a question in the GUI input box.
  • The input text is preprocessed (tokenization, normalization).
  • Semantic meaning is extracted and matched against stored patterns.
  • A relevant pre-defined response from the JSON knowledge base is returned.

Tech stack

  • Python (3.x)
  • NLTK for basic NLP
  • Tkinter for GUI
  • JSON for response patterns / dataset
