Chat Bot Using Python

By Abishek Prabakaran R • 2 min read • Jan 2, 2025

This project is based on Tkinter for the GUI, and uses a JSON file for the AI logic.

We propose a system that functions as an application providing users with information about their present health. The system determines what the user is inquiring about by retrieving the semantic meaning of the query. That semantic text is then used as input for a pattern-matching algorithm. The patching (pattern-matching) algorithm leverages pre-fed knowledge to generate a response.

The interface presents two distinct formats: one rendered via a Python GUI, and another through a Python-based JSON output. Both formats display code outputs reflecting chatbot responses for Coronavirus-related queries.
