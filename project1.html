Chat Bot Using Python

Authors: Abishek R

This project focuses on developing an AI-based chatbot using Python, aimed at answering queries related to the coronavirus (COVID-19). The bot uses Natural Language Processing (NLP) and Machine Learning techniques to respond to user input intelligently.

Abstract

The chatbot is designed to assist users by providing accurate information on COVID-19 symptoms, prevention measures, and general guidelines from WHO and other trusted sources. The goal is to reduce misinformation and improve accessibility of verified knowledge.

Features

  • Interactive text-based chatbot interface
  • Pre-trained dataset for COVID-19 related queries
  • Integration with NLP libraries
  • Lightweight and easy to deploy

Technology Stack

  • Python 3.x
  • NLTK (Natural Language Toolkit)
  • TensorFlow/Keras (for deep learning models)
  • Flask (for web deployment)
