Chatbot Using Python

This project demonstrates how to build a chatbot using Python. The chatbot can be used for answering COVID-19 related queries and providing information. The implementation uses NLP techniques and integrates with various datasets to improve accuracy.

View Project on GitHub View on ResearchGate

Overview

The chatbot was developed to assist users in retrieving relevant COVID-19 information efficiently. By leveraging Python libraries like NLTK and scikit-learn, the chatbot processes user queries and returns accurate responses.

Features

  • Text-based interaction
  • COVID-19 dataset integration
  • Natural Language Processing
  • User-friendly interface

Technology Stack

  • Python
  • NLTK
  • scikit-learn
  • Flask (for deployment)
