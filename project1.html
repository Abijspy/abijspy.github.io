Chatbot Using Python

This project demonstrates the creation of a simple chatbot in Python, designed specifically for answering queries about the Coronavirus. The bot utilizes Python libraries for natural language processing and pattern matching to respond accurately to user queries.

Project Overview

The chatbot was built with Python and leverages a combination of NLP techniques to interpret and respond to user questions about COVID-19. It provides instant, accurate answers based on predefined datasets and responses.

Features

  • Python-based backend
  • Natural language processing
  • COVID-19 specific knowledge base
  • Instant response time
