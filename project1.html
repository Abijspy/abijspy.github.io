Chat Bot Using Python

In this project, I have created a simple Chat Bot using Python that responds to user queries. It demonstrates how Natural Language Processing (NLP) can be implemented in Python to create conversational agents.

Video Demonstration

Project Overview

The chat bot is developed using Python libraries like NLTK and makes use of predefined patterns to respond to inputs. It simulates basic conversation and can be expanded to handle more complex queries.

Technologies Used

  • Python
  • NLTK Library
  • Regex for pattern matching

Features

  • Responds to basic user queries
  • Can be customized with additional patterns
  • Lightweight and easy to run

Source Code


import nltk
from nltk.chat.util import Chat, reflections

pairs = [
    [
        r"hi|hello|hey",
        ["Hello!", "Hi there!", "Hey!"]
    ],
    [
        r"how are you ?",
        ["I'm doing fine, thank you!", "I'm good, thanks for asking!"]
    ],
    [
        r"quit",
        ["Bye! Take care.", "Goodbye!"]
    ]
]

def chatbot():
    print("Hi! I'm your chatbot. Type 'quit' to exit.")
    chat = Chat(pairs, reflections)
    chat.converse()

if __name__ == "__main__":
    chatbot()
